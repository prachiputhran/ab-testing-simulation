# A/B Testing Simulation with Statistical Power Analysis

This project simulates a real-world A/B test using rigorous statistical methodology.

## 📌 Overview
We evaluate whether a change (e.g., CTA button) improves conversion rate using hypothesis testing and power analysis.

## 🧠 What this project covers
- Hypothesis testing (two-proportion z-test)
- Power analysis to determine sample size
- Simulation of user behavior (binomial trials)
- Effect size measurement (Cohen’s h)
- Monte Carlo validation (empirical power & Type I error)

## ⚙️ Tech Stack
- NumPy
- SciPy
- Statsmodels
- Matplotlib
- Seaborn
